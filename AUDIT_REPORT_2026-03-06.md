## Метод расчёта

Для каждого клиента берётся **последний лист** файла, где он присутствует (март → февраль → январь). Из этого листа берутся **все строки** клиента. Долг рассчитывается по колонке остатка:

```
Январь:  колонка AA (27) — «Ост на 31.01.2026»
Февраль: колонка AB (28) — «Ост на 28.02.2026»
Март:    колонка AB (28) — «Ост на 31.03.2026»
```

**Формула:**

```
Debt(клиент) = SUM(остаток) всех строк клиента в этом листе
TotalDebt    = SUM(Debt(клиент)) для всех клиентов где Debt > 0
```

Остатки разных месяцев **не суммируются**. Колонки M→AA (история платежей) **не используются** для расчёта. Единственный источник — колонка остатка.
